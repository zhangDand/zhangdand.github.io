hexo常用命令

发表于 2016-09-21   |   分类于 Hexo ， Command

记录一些hexo的常用命令

常用简写

  • hexo n 新建文章
  • hexo g 新建页面
  • hexo d 部署
  • hexo s 本地端口
  • hexo d -g 生成加部署
  • hexo s -g 预览加部署

    常用完整命令

  • hexo new "<filename>" 新建文章

  • hexo new page "<pagename>" 新建页面

  • hexo generate 生成静态页面至public目录

  • hexo server 开启预览访问端口

  • hexo deploy 将.deploy目录部署到GitHub
